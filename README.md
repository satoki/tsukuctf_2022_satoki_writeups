- Misc

メインのOSINTはいつも通りの旅行先で撮った写真を利用しましたが、ツイートしていないものを選ぶのが大変でした。  
MiscはSatoki色が出るようなものに、Webは恒例のGuessと中程度の難易度の問題です。  
ちなみに、MiscやWebは私をOSINTするとヒントが手に入ります。
